### Getting Started

1. **Clone the Repository:**

```bash
  git clone https://github.com/shivamsheokand/Day-3.git
```

Install Dependencies:

```bash
cd Day-3
npm install
```

Run the App:

```bash

npm start
```

View the App:

Open your browser and go to http://localhost:3000 to see the React home page.
